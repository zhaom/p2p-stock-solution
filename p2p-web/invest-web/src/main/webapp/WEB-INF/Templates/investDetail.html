投资后什么时候开始计算收益？
投资项目开始计算收益的时间是指658金融网将筹款资金放款至借款人操盘账户的时间。例如：658金融在2014年01月02日将借款人筹款资金放款至其操盘账户，则项目计算收益的时间为：2014年01月02日。
股票配资符合法律规定吗？
借钱人签署的合同，属于民间借贷合同，而作为民间借贷行为是合法的、有效的且受法律保护的。个人与个人之间的借贷，历来有之，不违法。
我在658金融的资金安全吗？
658金融的资金分为自有资金和客户资金，采用分账管理模式，自有资金和用户资金完全分离。用户充值和提现的资金的收付行为全部在具有央行颁发支付牌照的知名第三方支付平台贝付以及银行体系内操作。
什么是风险保证金？
借款人在申请融资时，需要交纳一笔风险保证金，凭风险保证金借款人可以申请一定比例的额度，平台上所有的投资项目都已交纳风险保证金，请放心投钱。
如何保证投资人的本金和收益？
借款人所使用股票账户均由658提供，658拥有账户管理权，借款人只能买卖，无法提现。且借款人每笔借款都需缴纳一笔保证金。如账户出现亏损，658将卖出股票，账户停止使用。每笔借款的本金及收益都将于项目结束后下一个工作日返还至投资者的账户。坐享收益，实现投资无风险。
