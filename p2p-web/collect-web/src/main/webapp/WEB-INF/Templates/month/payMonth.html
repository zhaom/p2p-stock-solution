为了控制风险，借款人承诺遵守如下限制性规定：

（1）不得购买权证、ST、*ST、S*ST、SST、分级基金以及被交易所特别处理的股票；

（2）不得投资首日上市新股、复牌首日股票等当日不设涨、跌停板限制的股票；

（3）不得投资跌停、连续跌停、连续跌停后首次打开跌停板的股票；

（4）不得买上一年亏损和近半年亏损的创业板股票；

（5）除经事先同意外，账户内的资金用于购买单个证券品种的比例不得超过50%(沪深300ETF基金除外)；或用于购买创业板单个证券品种的比例不得超过30%。因市值下降导致单个证券品种的比例超过规定的，借款人应该在超过比例的第二个交易日上午10:00前卖出该支股票降低至规定比例之下；

（6）单只股票的市值不得超过流通总市值的0.5%；

（7）单只股票持仓总市值不得超过该股前5个交易日日均成交额的20%。

（8）相关法律法规禁止从事的其他投资。

（9）如按照收盘价计算，借款人在某一交易日违反上述任一约定，则借款人须在下一个交易日上午10:00前自行调整至约定比例，借款人特别授权658金融网有权以任何可以成交的价格完全卖出违反本条约定之类别股票。如某一股票在借款人买入后，该股票或者发行该股票的公司发生上述限制性规定的任一情形，除非经658金融网另行书面同意，借款人应在该等情形发生后的下一交易日内卖出该股票。若借款人不遵守上述规定，借款人特别授权658金融网可以卖出不符合规定的股票，由此造成的一切损失均由借款人承担。

（10）若借款人持有的股票当天出现跌幅达到9.5%（即：距离跌停板为0.5%）时，借款人特别授权658金融网有权卖出该股票。
